## 电子版作业格式


### 提交格式：姓名-学号-第n次作业.rar

	包含：
	--code.file
	--readme #说明程序运行所需要的环境以及如何运行
	--第n次编程作业报告.doc


### 编程作业报告格式：

	1.问题描述
	2.数据结构：简要说明选择的数据结构
	3.问题分析
	4.设计算法
	5.正确性证明
	6.算法分析：复杂度分析
	7.设计程序：程序用x语言实现，见code.file
